/Computacion grafica - Tranformadas con teclas/

¨NOTA:los cubos apareceran superpuestos, al momento de aplicar las transformadas, se aprecian cada uno.

- Presione las tecla 1 - De la parte superior del teclado, para generar el primer cubo.

- Presione las tecla 2 - De la parte superior del teclado, para generar el segundo cubo.

- Presione la tecla Z para escalar el segundo cubo y la tecla X para trasladarlo.

- Presione las tecla 3 - De la parte superior del teclado, para generar el tercer cubo.

- Presione la tecla C para escalar el tercer cubo y la tecla V para trasladarlo.

- Presione la tecla F para finalizar y reiniciar la escena.
